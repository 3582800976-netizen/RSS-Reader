## macOS 一键安装（推荐）

> 直接下载 DMG，**无需**配置 Python / Node。适用于 **Apple Silicon** Mac。

**[下载 Mercury Web-0716-arm64.dmg（Latest Release）](https://github.com/3582800976-netizen/RSS-Reader/releases/latest)**

### 重要：从 GitHub 下载后首次打开

从网页下载的安装包未做 Apple 签名，系统可能提示「无法验证是否包含恶意软件」。**这是正常现象**，请：

1. 打开 DMG → 拖入 **应用程序**
2. 在应用程序文件夹中 **右键 Mercury Web → 打开 → 再次确认打开**
3. 首次放行后可双击启动；浏览器将自动打开阅读界面

> 若仅双击图标，macOS 可能直接拦截。详细说明见 [test0716/packaging/DISTRIBUTION.md](test0716/packaging/DISTRIBUTION.md)
