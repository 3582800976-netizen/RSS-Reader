
## 2. 已完成工作

### 2.1 内容清洗后端（必做②）

| 模块 | 说明 |
|------|------|
| `services/cleaning.py` | Readability 提取 → Bleach 白名单 → html2text 转 Markdown |

| 降级策略 | Readability 失败时回退 sanitize 后 HTML |

```text
ReaderPane → MarkdownBody → /api/entries/{id}/cleaned → react-markdown 渲染
```
